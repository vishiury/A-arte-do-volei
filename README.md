# A-arte-do-volei
Este é um projeto de programação pensado por Fábio Penha em parceria com Thiago Theiry.
O trabalho "A-arte-do-volei" gira em torno de um esporte que vem cada vez mais crescendo no Brasil, o vôlei. Esta modalidade não possui uma data de chegada expecifica registrada no Brasil, entretanto estima se que a sua chegada foi por volta dos anos 1915, com a primeira partida de vôlei no Nordeste, mas desde sua chegada houve grandes jogadores destaques. Este trabalho visa falar sobre os jogadores brasileiros.
  Tempo de profissão:
    O esporte é algo que pode tanto ser feito por um entusiasta, que de certa forma alegra quem exerce, ou como trabalho, que inspira e dá forças a outras pessoas, e tudo tem um           começo tão quão o início de exercer a profissão.
  salário:
    O salário é algo fundamental para um trabalhador, mas qual é a estimativa de  um jogador profissional de vôlei de alto rendimento?
  Treinos:
    Os jogadores tem que manter um físico atlético com uma rotina regrada de treinos específicos para o vôlei, mas como são esses treinos?
  Saúde mental:
    A saúde corporal é muito importante para um atleta, mas a saúde mental é tão importante quanto, Por isso merece cuidados expecíficos para ações realizadas no vôlei proficional.
  Títulos:
    Dentro do esporte, a necessidade de holofortes é obrigatória, e o jogador de vôlei precisa correr atrás de títulos para assim conseguir o prestígio que marca toda a sua carreira.
