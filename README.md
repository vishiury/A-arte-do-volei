# A-arte-do-volei
Este é um projeto de programação pensado por Fábio Penha em parceria com Thiago Theiry.
O trabalho "A-arte-do-volei" tem como Principal objetivo explicar sobre:
  O volei:
    Chegada no Brasi: O vôlei brasileiro não possui uma data expecífica registrada da sua chegada, entretanto estima se que a sua chegada foi por volta dos anos 1915, com a primeira partida de vôlei.
    Principais jogadores: Desde a chegada do voleibol no Brasil até os dias atuais, há uma coleção enorme de jogadores entretanto houve jogadores destaques como por exemplo o levantador                       "Ricardinho", o libero "Serginho" e ponteiro "Giba".
